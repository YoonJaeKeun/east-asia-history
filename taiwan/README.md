# 대만사 (Taiwan)

## 주요 시대

| 시대 | 기간 | 비고 |
|------|------|------|
| 선사·원주민 | 기원전 4000년경 ~ 17세기 | 오스트로네시아계 원주민, 신석기 문화 |
| 네덜란드·스페인 | 1624년 ~ 1662년 | 대항해시대, 한족 이주 시작 |
| 정씨 정권(동녕) | 1662년 ~ 1683년 | 정성공, 반청복명 기지 |
| 청 통치 | 1683년 ~ 1895년 | 한족 이민 사회 성장, 1885년 성 승격 |
| 일본 통치 | 1895년 ~ 1945년 | 식민 통치, 황민화 |
| 전후 중화민국 | 1945년 ~ | 2·28 사건, 국민정부 이전, 민주화 |

## 자료 수집 우선순위

1. 원주민(오스트로네시아) 사회와 문화
2. 대항해시대 해상 교역과 한족 이주
3. 청·일본 통치기의 사회 변동
4. 동아시아 해상 질서 속 대만의 지정학
