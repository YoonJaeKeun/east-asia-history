# 동아시아 역사 자료 (East Asia History)

동아시아(한국·중국·일본 등) 역사 자료, 노트, 연표를 정리하는 저장소입니다.

## 폴더 구조

```
east-asia-history/
├── README.md          # 이 문서
├── korea/             # 한국사
├── china/             # 중국사
├── japan/             # 일본사
├── taiwan/            # 대만사
├── okinawa/           # 오키나와·류큐사
├── timeline/          # 통합 연표
├── topics/            # 주제별 자료 (교류사, 전쟁, 사상 등)
└── references/        # 참고 문헌·자료 출처
```

## 정리 규칙

- 각 자료는 마크다운(`.md`)으로 작성합니다.
- 파일명은 영문 소문자 + 하이픈(`-`)을 사용합니다. 예: `goryeo-dynasty.md`
- 날짜·연도는 가능하면 구체적으로 기재합니다 (예: 기원전 108년, 1392년).
- 출처는 각 문서 하단 또는 `references/`에 정리합니다.

## 협업 방식

- 먼저 각국 개요와 공통 시기를 정리합니다.
- 그 다음 교류사와 비교 주제를 연결하여 문서를 확장합니다.
- 수정 이력은 각 파일 상단에 간단히 기록하거나 `README.md`에 업데이트합니다.
- 필요한 경우 `topics/`에 추가 문서를 만들어 보강합니다.
