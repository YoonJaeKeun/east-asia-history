# 오키나와·류큐사 (Okinawa / Ryukyu)

## 주요 시대

| 시대 | 기간 | 비고 |
|------|------|------|
| 선사(패총·구스쿠) | 기원전 ~ 14세기 | 어로·채집, 12세기 농경·구스쿠 축조 |
| 삼산 시대 | 14세기 | 호쿠잔·주잔·난잔 분립, 명에 조공 |
| 류큐 왕국 | 1429년 ~ 1879년 | 쇼하시 통일, 대교역 시대, 양속 체제 |
| 류큐 처분·오키나와현 | 1872년 ~ 1945년 | 왕국 멸망, 일본 편입과 동화 |
| 전후 | 1945년 ~ | 오키나와 전투, 미군 통치, 1972년 복귀 |

4. 류큐 처분과 근대 오키나와의 정체성
