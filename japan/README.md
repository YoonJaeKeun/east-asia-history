# 일본사 (Japan)

## 주요 시대

| 시대 | 기간 | 비고 |
|------|------|------|
| 야요이 | 기원전 300년경 ~ 3세기 | 초기 농경, 고분 문화 |
| 고훈 | 3세기 ~ 7세기 | 중앙집권화 시작, 야마토 정권 |
| 아스카 | 538년 ~ 710년 | 불교 수용, 율령 국가 |
| 나라 | 710년 ~ 794년 | 헤이조쿄 수도, 중앙 관료제 강화 |
| 헤이안 | 794년 ~ 1185년 | 궁중문화, 겐페이 전쟁 |
| 가마쿠라 | 1185년 ~ 1333년 | 무사 정권, 막부 성립 |
| 무로마치 | 1336년 ~ 1573년 | 센고쿠 시대, 문화 확산 |
| 아즈치모모야마 | 1573년 ~ 1603년 | 전국 통일, 유럽과 교류 확대 |
| 에도 | 1603년 ~ 1868년 | 쇄국, 도시문화 발달 |
| 메이지 이후 | 1868년 ~ | 근대화, 제국주의 시대 |

## 정리 계획

- 각 시기별 주요 사건 정리
- 중국·한국과의 교류 관계 기록
- 비교용 키워드: 정치, 문화, 외교, 군사, 경제
- 일본 고유의 문화적 흐름과 일본식 변용을 이해

## 파일 구성

- `README.md` : 일본사 전체 개요
- `ancient.md` : 고대 시대(야요이~나라)
- `medieval.md` : 중세 시대(헤이안~무로마치)
- `early-modern.md` : 근세 시대(아즈치모모야마·에도)
- `liquor.md` : 외전 — 일본 술의 역사(사케·쇼추, 그리고 와인·위스키)
- 이후 문서 예시 : `modern.md`
- `sources.md` : 참고 문헌과 자료 출처

## 문화 이해

- 일본 각 시대의 예술·문학·종교 특징을 정리합니다.
- 한자문화권 안에서 일본이 수용하고 변형한 요소를 분석합니다.
- 해외 교류와 쇄국의 문화적 영향을 비교합니다.

## 자료 수집 우선순위

1. 한자 문화권 내부 교류
2. 불교·유교 전파와 적응
3. 외국 세력과의 만남 (조선, 중국, 포르투갈)
4. 각국 시대 구분의 공통점과 차이점
