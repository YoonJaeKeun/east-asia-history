## 정리 계획

- 왕조별 주요 변화를 시기별로 정리합니다.
- 동아시아 공통 시기와 중국 왕조를 비교합니다.
- 한국·일본과의 외교 및 문화 교류를 중심으로 사례를 모읍니다.

## 파일 구성

- `README.md` : 중국사 전체 개요
- `ancient.md` : 전사(신석기~서주)와 고대 시대(춘추전국~위진남북조)
- `medieval.md` : 중세 시대(오대십국~송·원)
- `early-modern.md` : 근세 시대(명·청)
- `liquor.md` : 외전 — 중국 술의 역사(신석기 발효 음료~백주)
- 이후 문서 예시 : `modern.md`
- `sources.md` : 참고 문헌과 자료 출처

## 문화 이해

- 각 왕조의 주요 문화·사상·예술 흐름을 정리합니다.
- 유교·도교·불교 등 사상이 사회에 미친 영향을 살핍니다.
- 중국 문화가 동아시아에 끼친 확산과 변용을 분석합니다.
